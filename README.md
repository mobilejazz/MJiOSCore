# MJiOSCore

## How to 

To install the MJiOSCore library, just paste the following line in your podfile:
```
pod 'MJiOSCore', :git => 'https://github.com/mobilejazz/MJiOSCore.git', :tag => '0.1.0'
```

## Dependencies
- Haneke
- Cloudinary
- UIImage+Additions
- UIColor+Additions
